# Weavencraft (Eaglercraft 1.21.11 + Fabric)

Weavencraft is a modular modded Minecraft server platform built for Eaglercraft clients and Fabric-based server templates.

Originally based on a setup by wadwdwd (Discord: wadwdwd / GitHub: wadwdwd1), this project has been modified and expanded into a multi-template system.

---

⚠️ THIS PROJECT IS STILL IN DEVELOPMENT  
This README will be updated as the server system and modular version templates are expanded.

Planned feature: a modular template system that allows swapping different server/version packs without modifying core startup logic.

---

## ⚠️ IMPORTANT MOD INFO

All mods must be compatible with:

- Fabric
- The specific Minecraft version of the selected server template
- PolyMc / PolyMc-extra (for compatibility translation)

---

## 🧠 EXTRA INFO

1. The server runs through a Velocity proxy using a direct IP connection.  
2. The system can be used as a private/single-player-style experience if no other players join.  
3. Some features may behave differently depending on client type (Eagler vs Java Edition).  

---

## 📦 TEMPLATE SYSTEM (IN DEVELOPMENT)

Weavencraft uses a modular template system that allows different gameplay and version configurations.

You will be able to:

- Switch between prebuilt server/version templates  
- Enable different gameplay packs (QoL, Horror, Experimental)  
- Test mods separately before applying them to main gameplay builds  

Each template is a fully isolated server environment and may include:

- A specific Fabric version (within the supported range)  
- A curated modpack  
- A specific gameplay theme or mechanics set  

---

⚠ Experimental templates may be unstable or incompatible with some clients.
